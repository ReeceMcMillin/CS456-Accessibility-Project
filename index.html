Letterboxed

Try to solve the puzzle in 5 words!

Instructions

  1. Use TAB to cycle between elements.
  2. To select a letter, you can use one of the following two options.
    • Press the SPACE key while the letter is selected.
    • Press that letter on your keyboard.
  3. You can't select two letters in a row from the same side.
  4. To finish a word, press ENTER.

Goals

The primary goal of this project is to port the New York Times game Letter Boxed to a form more friendly to users with limited sight or visual processing issues.

  • Screen reader accessibility
  • Easily-differentiable letters via the Atkinson Hyperlegible font.
